# Refleksi Praktikum DevOps

Praktikum DevOps ini memberikan saya pemahaman awal mengenai pentingnya
lingkungan pengembangan yang terstandarisasi. Saya menyadari bahwa
instalasi tools seperti Git, Docker, dan Visual Studio Code bukan hanya
sekadar memenuhi tugas, tetapi merupakan fondasi penting dalam dunia
kerja sebagai software engineer.

Saya berharap melalui mata kuliah DevOps ini saya dapat memahami konsep
CI/CD, containerization, serta deployment aplikasi secara otomatis.
Selain itu, saya ingin menguasai penggunaan Docker secara lebih mendalam
dan memahami bagaimana pipeline deployment bekerja di industri nyata. Di
akhir semester, saya berharap mampu membangun dan mendeploy sebuah
aplikasi menggunakan prinsip DevOps secara mandiri.
